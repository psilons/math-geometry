## Latex Introduction

command: 
```\command[optional arguments]{argument1}{argument2}```
for example
```\document[11pt, a4paper]{article}```

special characters need to be escaped by \, such as # $ & _ { } 

font shapes: \textbf, \textit, \textsl, \textsc, \emph, 

fonts: \textsf - serifs font
       \textfm - Roman font
